📦 Docker Labs

A practical learning repository for Docker where I build and experiment with real-world applications using JavaScript (Node.js) and Python (Flask/FastAPI).

🎯 Objective

This repository is focused on understanding containerization step by step, starting from basic concepts and gradually moving toward multi-tier application architectures used in real systems.

🧱 What This Repo Covers
Basic Docker usage and container lifecycle
Running and managing containers
Building simple static and backend applications
Containerizing JavaScript and Python apps
Understanding service-based architecture (frontend, backend, database)
Moving toward real-world multi-tier systems
🛠️ Tech Stack
JavaScript Side
Node.js
Express.js
Python Side
Python
Flask / FastAPI
Infrastructure
Docker
Nginx (for serving and routing in later stages)
🧪 Project Levels
Static website containerization
Basic Linux containers for practice
Single backend services (Node.js / Python)
Two-tier applications (frontend + backend)
Three-tier applications (frontend + API + database)
📌 Goal of This Repo

To build strong hands-on understanding of Docker and prepare for real-world DevOps workflows by working with modern application stacks.

🚀 Future Expansion

This repository will gradually expand into:

Docker Compose setups
CI/CD pipelines
Microservices architecture
Kubernetes integration
Cloud deployment practices
🧠 Learning Focus

The goal is not just to run containers, but to understand how real applications are structured, connected, and deployed using containerization.
